歌单文件说明

- `#` 是注释
- `,` 分割 BV 号和歌名，一个 BV 号含多首歌，也用该符号分割，如 `BVxxxxx, 歌名1, 歌名2`
- `/` 歌名有多种写法，用该符号分割，如 `BV18v4y1M7EQ, 【阿梓歌】《寓言+引路的风筝》国风纯享/寓言 + 引路的风筝 (Live), 寓言, 引路的风筝`
- `\\` 和 `,,` : 如果歌名中含有 `,` 或 `/`，用这两个符号替代，例如 `BV1ru4y127St,, 天使にふれたよ!  / 相遇天使 🎸\\天使にふれたよ!\\相遇天使`
